# Personal Portfolio

### Welcome to my portfolio website! This project is built using React.js and Framer Motion to showcase my skills, projects, and experiences in an interactive and visually appealing manner.

## Features

- **Responsive Design:** The website is designed to be fully responsive, ensuring optimal viewing experience across a wide range of devices and screen sizes.
- **Interactive Navigation:** Navigation elements are animated using Framer Motion, providing smooth transitions and engaging user interactions.
- **Project Showcase:** A dedicated section highlights my projects, allowing visitors to explore each project's details, technologies used, and live demos.
- **Skills Section:** An overview of my skills and expertise is provided, giving visitors insight into my capabilities and proficiencies.
- **About Me:** A brief introduction to who I am, my background, and what I'm passionate about is included to personalize the experience for visitors.
- **Contact Form:** A contact form is available for visitors to reach out to me directly, making it easy to connect and collaborate.

## Technologies Used
- React.js
- Framer Motion
- JavaScript
- TailwindCSS
- React-router
- Vite
- slick-carousel
